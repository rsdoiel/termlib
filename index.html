termlib

A minimalist terminal display library written as a Go module. It explores the space between fmt package and a rich library like tcell.

Release Notes

  • version: 0.0.7

  • status: concept

  • released: 2026-05-12

  • Ctrl+J (0x0a) now inserts into the buffer and moves to a new visual line with "… " continuation prompt. Enter ( is the only submit key.

  • redraw was refactored to be line-aware: it draws only the current visual line (from the last in buf to the next or end), choosing prompt for line 1 and "… " for subsequent lines.

  • Backspace across a clears the current visual line ([K), moves up (\033[1A), and redraws the merged previous line.

  • Left/Right arrows, Home/End, Ctrl+A/E/K are all clamped to the current line — they won't jump across boundaries.

  • History navigation (Up/Down) is disabled when lineCount > 0.
