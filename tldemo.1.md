# NAME

tldemo

# SYNOPSIS

tldemo [OPTIONS]

# DESCRIPTION

**tldemo** demonstrate the simple terminal UI that can be implemented with termlib. This is a bare bones approach not a full TUI package like tcell.

# OPTIONS

-help,
: Display this help

-version
: Display tldemo version

-license
: Display tldemo license
